# my react-webpack-redux-router-reactstrap-mocha-babel-setup

## Features

* React 16
* Webpack 4
* Babel
* Mocha
* Reactstrap
* Redux
* Thunk
* Staging ES Next Features
* Hot Module Replacement

## Installation

* `git clon repoe`
* cd repo
* npm install
* npm start
* visit `http://localhost:8080/`
